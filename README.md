# Gauche-net-oauth

This module provides an interface OAuth authentication.

## Install:

   From source:

     $ git clone git@github.com:mhayashi1120/Gauche-net-oauth.git
     $ cd Gauche-net-oauth
     $ ./configure
     $ make
     $ make -s check
     $ [sudo] make install

   ('-S root' option or 'sudo' may be required if you want to install
   the package system-wide.)
